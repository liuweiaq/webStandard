总体要求

  • HTML页只写入文档结构，样式应写在CSS文件中；
  • 所有CSS文件统一采用UTF-8无BOM编码；
  • CSS代码单词一律小写；
  • 框架引入：只能引入Bootstrap作为前端CSS框架，辅助开发。如有特别需求，须与团队其他成员讨论决定；
  • CSS选择器一般使用class选择器和标签选择器，不推荐使用id选择器。

    • CSS命名规范

    文件命名规范

  • CSS文件名及作用：master.css(主样式)、layout.css(框架布局样式)、font.css(字体相关样式)、print.css(打印相关样式)；
  • CSS规则根据其作用，分别存储于以上CSS文件中；
  • 不能随意更改以上文件名称；
  • 一般不再新增CSS文件，如有特别需求，须与团队其他成员讨论决定。

    • 类(class)命名规范

  • 所有类名均须加前缀oso，如：oso-header；
  • 类名单词之间用中横线(-)连接，如：oso-main-nav；
  • 用于命名的英语单词应能准确概括该规则的内容或功能，不能使用样式信息、汉语拼音、不常见词汇等进行命名。下表列出了一些常用单词可供参考：
    • 布局类
    header(页头)、footer(页脚)、main(页面主体)
    top(顶部)、bottom(底部)、center(中间)、left(左侧)、right(右侧)
    inner(内部)、outer(外部) 、wrapper(外套样式)
    sidebar(侧栏)、content(内容)
    column(列)、row(行)
    导航类
    nav(导航)、menu(菜单)、title(标题)、summary(摘要)
    功能类
    tab(标签)、icon(图标)、scroll(滚动)、sub(子)、logo(标志)、link(链接)、banner(广告条)、status(状态)、toolbar(工具条)、register(注册)、guild(指南)、news(新闻)

    示例：

    Bad Code Good Code
    .oso-left /*使用了样式信息*/
    .oso-waitao /*使用了汉语拼音*/     		.oso-sidebar
    .oso-wrapper

    CSS代码书写规范

    基本书写规范

  • 代码缩进的基本单位为4个空格(或一个tab键)。
  • 每一个声明必须以分号结尾。
    • Bad Code Good Code
    .oso-main-nav{
         width:95% /*声明后缺少分号*/
    }     		.oso-main-nav{
         width:95%;
    }
  • 所有声明都要分行排版。
    • Bad Code Good Code
    .oso-main-nav{width:95%;height:30px; } .oso-main-nav{
         width:95%;
         height:30px;
    }
  • 多个选择器共用样式集时，选择器名称须换行排版。
    • Bad Code Good Code
    .oso-main-nav, .oso-main-sub{
         ……
    }
         		.oso-main-nav,
    .oso-main-sub{
         ……
    }
  • 定义尺寸属性时，必须添加单位(0值除外)。
    • Bad Code Good Code
    .oso-button{
         margin:10 0px 5;
    }     		.oso-button{
         margin:10px 0 5px;
    }
  • 建议属性编写顺序：
    • 顺序 属性
    1 位置属性(position, top, right, z-index, display, float等)
    2 大小(width, height, padding, margin)
    3 文本相关(font, line-height, letter-spacing, color- text-align等)
    4 背景(background, border等)
    5 其他(animation, transition等)

    示例：

    Bad Code Good Code
    .oso-example{
         color:red;
         z-index:-1;
         background-color:#9e0;
         display:inline-block;
         font-size:1.5em;
    }     		.oso-example{
         z-index:-1;
         display:inline-block;
         font-size:1.5em;
         color:red;
         background-color:#9e0;
    }
  • 建议选择器的嵌套层级不大于3级，位置靠后的限定条件应尽可能精确。
    • Bad Code Good Code
    .page .header .login #username input {
         ……
    }
    .comment div * {
         ……
    }     		#username input {
         ……
    }
    .comment .avatar {
         ……
    }

    说明：以上示例中使用了id选择器和通用选择符，这两种选择符在CSS代码中不推荐使用，这里引入只是为了方便举例。

    代码注释

  • CSS代码注释有两种样式：

    • ①单行注释

      					
  					/*注释内容*/

    ②块注释

      					
  					/*===================
    
					*第一行注释内容
    
					*第二行注释内容
    
					====================*/
  • 文件、模块、CSS规则等的注释，在其上一行使用块注释；声明的注释，在当前行尾使用单行注释。如下例所示：
    •   					
  					/*===================
    
					*CSS规则注释内容
    
					====================*/
    
					.oso-main-nav{
    
					    	width:100%;
    
					    	height:30px;
    
					    	background:url(images/mainMenu_bg.jpg) repeat-x;   /*声明注释内容*/
    
					}

    属性缩写

    CSS中，尽量使用缩写属性代替全写。这样不但能精简代码，同时又能提高用户阅读体验。常见CSS属性缩写如下表所示：

    属性 全写 缩写
    颜色 #000000
    #336699     		#000
    #369
    盒尺寸
    (margin/padding)     		padding-bottom:2px;
    padding-left:1px;
    padding-right:1.2px;
    padding-top:0;     		/*边框顺序是上右下左*/
    Padding:0 1.2px 2px 1px;
    边框 border-width:1px;
    border-style:solid;
    border-color:#000;     		/*语法:border:width style color;*/
    border:1px solid #000;
    背景 background-color:#f00;
    background-image:url(background.gif);
    background-repeat:no-repeat;
    background-attachment:fixed;
    background-position:0 0;     		/*语法:background:color image repeat attachment position;*/
    background:#f00 url(background.gif) no-repeat fixed 0 0;
    字体 font-style:italic;
    font-variant:small-caps;
    font-weight:bold;
    font-size:1em;
    line-height:140%;
    font-family:"LucidaGrande",sans-serif;     		/*语法:font:style variant weight size/line-height family*/
    font:italic small-caps bold 1em/140% "Lucida Grande",sans-serif;
    列表 list-style-type:square;
    list-style-position:inside;
    list-style-image:url(image.gif);     		/*语法:list-style:type position image*/
    list-style:square inside url(image.gif);

    说明：更多属性写法可以参考：CSS参考手册。
